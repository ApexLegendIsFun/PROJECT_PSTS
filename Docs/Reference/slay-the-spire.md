# Slay the Spire Reference

## Overview
PROJECT_PSTS 전투 시스템의 주요 레퍼런스. 기본 구조를 차용하되 TRIAD 시스템으로 차별화.

---

## StS vs PROJECT_PSTS 비교

| 항목 | Slay the Spire | PROJECT_PSTS |
|------|----------------|--------------|
| **캐릭터** | 1명 | 3명 (TRIAD) |
| **덱** | 공유 1개 | 개별 3개 |
| **에너지** | 공유 (3) | 캐릭터별 개별 |
| **HP** | 1개 | 캐릭터별 개별 |
| **드로우** | 5장/턴 | 각자 N장/턴 |
| **행동** | 자유 (턴 내 무제한) | 순차 (A→B→C) |
| **턴 구조** | 플레이어 → 적 | 플레이어 → 적 (동일) |

---

## StS에서 채용하는 요소

### 턴 구조
- 플레이어 턴 → 적 턴 반복
- 턴 시작 시 드로우 + 에너지 회복
- 턴 종료 시 버리기 + Block 초기화

### Intent 시스템
- 적의 다음 행동 미리 표시
- 전략적 의사결정 유도

### 카드 풀 순환
- Draw → Hand → Discard → Shuffle → Draw
- 덱 관리의 중요성

### Block 메카닉
- 방어도로 데미지 흡수
- 턴 종료 시 초기화

### 상태효과 기본 구조
- Strength/Dexterity (공격/방어 버프)
- Vulnerable/Weak (디버프)
- Poison 등 DoT

---

## StS와 차별화하는 요소

### TRIAD (3캐릭터)
- **복잡도 증가**: 3개 덱 관리
- **전략 다양화**: 캐릭터 조합/시너지
- **리스크 분산**: 한 캐릭터 사망해도 게임 계속

### 개별 에너지
- 캐릭터별 자원 관리
- 한 캐릭터에 집중 vs 분산 전략

### 순차 행동
- 행동 순서의 전략적 의미
- 버프 → 공격 순서 고려

## StS 핵심 밸런스 수치 (참고용)

```
기본 에너지: 3
기본 드로우: 5장
기본 HP: 70-80 (캐릭터별)

Strike (기본 공격): 1에너지, 6데미지
Defend (기본 방어): 1에너지, 5블록

Vulnerable: 받는 데미지 50% 증가
Weak: 주는 데미지 25% 감소
```
